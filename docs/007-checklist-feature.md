# 007: チェックリスト機能

## 概要
カードにチェックリスト（ToDo）を追加する機能を実装する。

## 関連要件
- 5-2. カード機能 - チェックリスト

## 実装内容

### 1. チェックリストUI
- [x] チェックリストコンポーネント作成
- [x] チェックボックス + テキスト表示
- [x] チェック状態のトグル（ハプティクス付き）
- [x] 完了項目のスタイル（取り消し線）

### 2. チェックリスト編集
- [x] 項目追加UI（テキスト入力 + 追加ボタン）
- [x] 項目テキスト編集（タップで編集モード）
- [x] 項目削除（長押しで確認ダイアログ）
- [ ] 項目の並び替え（将来実装）

### 3. カード詳細での統合
- [x] カード詳細画面にチェックリストセクション追加
- [x] 進捗表示（例: 2/5 完了）
- [x] 常にセクション表示（空でも追加可能）

### 4. ストア連携
- [x] チェックリスト更新処理
- [x] 永続化（Zustand + AsyncStorage）

## 完了条件
- [x] カードにチェックリストを追加できる
- [x] チェック状態をトグルできる
- [x] 項目を追加・編集・削除できる
- [x] 進捗が表示される

## 備考
- シンプルなUIを優先
- 項目数の上限は設けていない
- 長押しで削除の操作説明はオンボーディングで追加可能

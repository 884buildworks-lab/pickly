# 005: 共有インテント受信機能

## 概要
他アプリからの「共有」でURLを受信し、カードとして保存する機能を実装する。

## 関連要件
- 5-2. カード機能 - URL保存（共有から受信）
- 5-3. 保存フロー

## 実装内容

### 1. 共有インテント設定
- [ ] `app.json` の intent-filter 設定（Android）
- [ ] `app.json` の share extension 設定（iOS）
- [ ] Expo Linking の設定

### 2. 共有受信ハンドラー
- [ ] `hooks/use-share-intent.ts` - 共有受信フック
- [ ] URLの抽出処理
- [ ] アプリ起動時の共有データ確認
- [ ] バックグラウンドからの復帰時の処理

### 3. 保存モーダルの自動表示
- [ ] 共有受信時に保存モーダルを自動表示
- [ ] URLとメタデータの自動入力
- [ ] コレクション選択（直前利用をデフォルト）
- [ ] メモ入力フィールド

### 4. 保存フロー最適化
- [ ] 最小タップで保存完了
- [ ] 直前のコレクションをデフォルト選択
- [ ] ハプティクスフィードバック
- [ ] 保存完了のトースト表示

## 完了条件
- [ ] 他アプリから「共有」でURLを受信できる
- [ ] 受信時に保存モーダルが自動表示される
- [ ] URLとメタデータが自動入力される
- [ ] 直前のコレクションがデフォルト選択される
- [ ] 最小タップで保存が完了する

## 備考
- Expo Goでの動作確認に制限がある可能性
- Development Buildでのテストを推奨
- iOSとAndroidで実装が異なる場合がある
